USER WORKFLOW
================================================================================

1. Start application at http://localhost:5000
2. Enter Discogs API Consumer Key and Secret
3. Click "Authenticate" to generate OAuth token
4. View list of collection folders
5. Select a folder to process
6. Browse releases with sorting options
7. Select individual releases or "Select All"
8. Review CSV preview in table format
9. Click "Generate CSV" to download final file

================================================================================
QR FACTORY 3 COMPATIBILITY
================================================================================

The generated CSV uses placeholders that QR Factory 3 replaces with actual data:
- {artist} → Artist name
- {title} → Release title
- {url} → Discogs URL

Format: "Artist: {artist}\nTitle: {title}\nURL: {url}" in BottomText column
